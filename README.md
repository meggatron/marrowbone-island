# Marrowbone Island
a Python text adventure game 
Author: Meghan Thréinfhir

## Open in Binder
[![Binder](https://mybinder.org/badge_logo.svg)](https://mybinder.org/v2/gh/meggatron/marrowbone-island/HEAD)

This will open the project in a JupyterLab environment.  
To play the game:

1. Open the file `marrowbone_island.py` from the file browser on the left.
2. Click the **Play** button in the menu bar to run the script,  
   or open a terminal and run:

   ```bash
   python marrowbone_island.py
